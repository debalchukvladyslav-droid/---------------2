Ключі для AI (OpenRouter/Gemini) і Polygon через Supabase Edge Functions
==============================================================

Де взяти ключі
--------------
• Gemini (Google AI):
  https://aistudio.google.com/apikey
  Увійдіть у Google, натисніть Create API key, скопіюйте рядок.

• OpenRouter:
  https://openrouter.ai/keys
  Створіть ключ. Для цього проєкту рекомендовано зробити OpenRouter основним AI-провайдером.

• Polygon.io (хвилинні свічки, вкладка «Угоди»):
  https://polygon.io/dashboard
  Зареєструйтесь або увійдіть, далі Dashboard - API Keys - створіть ключ.


Налаштування в консолі Supabase
--------------------------------
1. Відкрийте https://supabase.com/dashboard і виберіть проєкт.

2. Зліва внизу: Project Settings - Edge Functions.

3. У блоці Secrets додайте (Add new secret):
   - GEMINI_API_KEY     = ваш ключ з Google AI Studio
   - OPENROUTER_API_KEY = ваш ключ з OpenRouter
   - AI_PROVIDER        = openrouter
   - OPENROUTER_MODEL   = openai/gpt-5.5 (або інший model id з OpenRouter)
   - POLYGON_API_KEY    = ваш ключ з Polygon
   - YOUTUBE_API_KEY    = (опційно) ключ Google Cloud / YouTube Data API v3 для блоку «Навчання»

   Змінні SUPABASE_URL і SUPABASE_ANON_KEY для функцій підставляються автоматично.


Деплой функцій (термінал, Supabase CLI)
----------------------------------------
  supabase login
  supabase link --project-ref <ваш_project_ref>

  supabase secrets set AI_PROVIDER=openrouter OPENROUTER_API_KEY=... OPENROUTER_MODEL=openai/gpt-5.5 POLYGON_API_KEY=... YOUTUBE_API_KEY=... APP_ALLOWED_ORIGINS=https://traderjournal-six.vercel.app,http://127.0.0.1:8787

  supabase functions deploy gemini-proxy
  supabase functions deploy polygon-aggs
  supabase functions deploy youtube-search


Як це працює в коді
--------------------
• Клієнт викликає POST .../functions/v1/gemini-proxy з Authorization: Bearer (сесія).
• Клієнт викликає POST .../functions/v1/polygon-aggs з тим самим токеном.
• Клієнт викликає POST .../functions/v1/youtube-search для пошуку відео в «Навчанні».
• Ключі OpenRouter/Gemini, Polygon і YouTube не потрапляють у браузер — лише секрети на Edge.

• gemini-proxy підтримує два AI-провайдери:
  - AI_PROVIDER=openrouter -> використовує OpenRouter
  - без AI_PROVIDER або AI_PROVIDER=gemini -> використовує Gemini

• Якщо gemini-proxy ще не задеплоєна (відповідь 404), AI піде на резервний шлях
  /api/gemini (наприклад Vercel), якщо він налаштований.

• Якщо YOUTUBE_API_KEY немає, у «Навчанні» показуються лише посилання на пошук YouTube
 (без превʼю відео).


Перевірка
---------
Після деплою увійдіть у журнал і перевірте AI або графік угоди.
Текст помилки про відсутній секрет — додайте змінну в Secrets і задеплойте знову.
